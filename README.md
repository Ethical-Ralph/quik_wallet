# Quik Wallet

A simple demo wallet system in Go, using dependency injection methodology .

## Installation

Quik wallet is written using the Go programming language so make sure you have Go installed.

##### Install project dependencies

```shell
go get
```

##### Set up ENV config

DATABASE_URL=
REDIS_ADDRESS=
REDIS_PASSWORD=

## Start

```shell
go run main.go
```

## Run Tests

```shell
go test
```
